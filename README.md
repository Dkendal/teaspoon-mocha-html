# teaspoon-mocha-html

https://www.npmjs.com/package/teaspoon-mocha-html

`yarn add teaspoon-mocha-html`

add

```js
//= require teaspoon-mocha-html/dist/index
```

or

```js
import 'teaspoon-mocha-html';
```

Replaces your default HTML reporter for teaspoon-mocha.
